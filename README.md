# Tandemo

Tandemo is a premium, real-time shared financial operating system designed for households and partners. Built with Next.js, Supabase, and Framer Motion, it provides a high-fidelity, Apple-inspired experience for managing shared finances.

## Features

- **Multi-User Sync**: Real-time synchronization via Supabase for household collaboration.
- **Active Allowance**: Track and manage shared allowances with a polished mobile-first UI.
- **Financial Dashboard**: Monarch-inspired dashboard with professional light-mode aesthetic.
- **AI-Driven Intelligence**: Automated budgeting and expense tracking.

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: Next.js
- **Database/Auth**: Supabase
- **Styling**: Tailwind CSS / Vanilla CSS
- **Animations**: Framer Motion
- **State Management**: Zustand
